费用报销管理
请求格式
URL/app.php/keep/payMonLst?token=token
HTTP请求方式POST
是否登录验证是
参数说明
字段名 参数名 必填 参数类型 说明
token token 是 string
分页页码 page 否 int
模糊搜索(申请日期、金额、说明) name 否 string
正常返回结果
字段名 变量名 必填 类型 说明
返回信息 msg 是 string
返回状态码 code 是 int
列表 results 是 object { "id": "id", "username": "申请人", "paymoney": "金额", "adddt": "申请日期", "status": "状态(不为3的都是未报销)", "paystatus": "报销说明" }
错误返回结果
字段名 变量名 必填 类型 说明
返回信息 msg 是 string
返回状态码 code 是 int
