分类名称搜索 searchname 否 string 客搜索供应商名称、单号、操作人、操作时间

字段名 变量名 必填 类型 说明

列表信息Tips:订单编号和订单名称换成供应商和采购单 results 是 object { id: "id", buldcom: "供应商名称", billnum: "采购单编号", optname: "采购人", optdt: "采购时间", sign: "是否检验", url: "未检验的，h5新增地址" }
