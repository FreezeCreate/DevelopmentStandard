维修单管理-统计条目
请求格式
URL/app.php/keep/serviceCount?token=token
HTTP请求方式GET
是否登录验证是
参数说明
字段名 参数名 必填 参数类型 说明
token token 必填 参数类型 说明
正常返回结果
字段名 变量名 必填 类型 说明
返回信息 msg 是 string
返回状态码 code 是 int
统计待处理的维修单 count 是 int 当没有待处理的维修单时，count=0
待办消息红点 todo_status 是 int todo_status=1表示有红点、todo_status=0表示没红点
错误返回结果
字段名 变量名 必填 类型 说明
返回信息 msg 是 string
返回状态码 code 是 int
