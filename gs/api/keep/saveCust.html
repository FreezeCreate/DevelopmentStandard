客户添加
请求格式
URL/app.php/keep/saveCust?token=token
HTTP请求方式POST
是否登录验证是
参数说明
字段名 参数名 必填 参数类型 说明
客户id id 否 int 修改客户信息时必传，添加客户传空或不传
性别 sex 是 string {'男','女'}
年龄 age 是 string
姓名 cust_name 是 string
公司 custcname 是 string
手机号 phone 是 string
地址 address 是 string
备注说明 info 是 string
正常返回结果
字段名 变量名 必填 类型 说明
返回信息 msg 是 string
返回状态码 code 是 int
错误返回结果
字段名 变量名 必填 类型 说明
返回信息 msg 是 string
返回状态码 code 是 int
