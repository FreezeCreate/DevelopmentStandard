请求格式

是否登录验证是

字段名 参数名 必填 参数类型 说明
审核id id 是 int id是详情页的bill对象里的id
审核状态status status 是 int 拒绝status=2;通过status=3
正常返回结果
字段名 变量名 必填 类型 说明
返回信息 msg 是 string
返回状态码 code 是 int
错误返回结果
字段名 变量名 必填 类型 说明
返回信息 msg 是 string
返回状态码 code 是 int
