客户上报
请求格式
URL/app.php/keep/saveService?token=token
HTTP请求方式POST
是否登录验证是
参数说明
字段名 参数名 必填 参数类型 说明
设备编号 number 是 string 查询设备是否存在,存在时页面展示设备相关信息，再添加；不存在提示设备信息有误，重新检测设备编号
服务类型 type 是 string {'维修','保养'}
备注说明 explain 是 string
正常返回结果
字段名 变量名 必填 类型 说明
返回信息 msg 是 string
返回状态码 code 是 int
用户登录状态 token 是 string 32位加密编码，保存用户登录状态用
错误返回结果
字段名 变量名 必填 类型 说明
返回信息 msg 是 string
返回状态码 code 是 int
