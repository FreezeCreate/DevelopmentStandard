付款类型列表
请求格式
URL/app.php/main/payType?token=用户token
HTTP请求方式GET
是否登录验证是
参数说明
字段名 参数名 必填 参数类型 说明
token token 是 string
分页 page 是 int
搜索 searchname 否 string
合同生效状态 distance 否 int 1、未生效2、生效中3、已过期
正常返回结果
字段名 变量名 必填 类型 说明
返回信息 msg 是 string 成功
返回状态码 code 是 int 0
列表信息 results 是 object { "id": "id", "name": "付款类型名称", "number": "编号", "status": "状态", "amount": "", "date": "添加时间", "type": "类型1、现金2、其他", "del": "是否删除1、是2、否" }
合同总金额 sum_all 是 int
合同总付款金额 sum_pay 是 int
合同未付金额 sum_not 是 int
错误返回结果
字段名 变量名 必填 类型 说明
返回信息 msg 是 string 失败
返回状态码 code 是 int 1
