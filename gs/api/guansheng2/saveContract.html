添加合同
请求格式
URL/app.php/custmang/saveContract?token=用户token
HTTP请求方式JSON
是否登录验证POST
参数说明
字段名 参数名 必填 参数类型 说明
审核通过的合同接口id apply_id 是 int （合同申请列表接口）
审核通过的合同接口名称 apply_name 是 string （合同申请列表接口）
说明 explain 否 string
合同金额 money 是 string
生效日期 startdt 是 date
截止日期 enddt 是 date
合同内容 content 是 string
签约日期 signdt 是 date
手机 phone 否 string
附件id files 否 string 附件id（上传文件接口），多个用“,”隔开
正常返回结果
字段名 变量名 必填 类型 说明
返回信息 msg 是 string 成功
返回状态码 code 是 int 0
错误返回结果
字段名 变量名 必填 类型 说明
返回信息 msg 是 string 失败
返回状态码 code 是 int 1
