新增工作计划
请求格式
URL/app.php/dailywork/saveDailyWork?token=用户token
HTTP请求方式JSON
是否登录验证POST
参数说明
字段名 参数名 必填 参数类型 说明
token token 是 string 5523cbad2881cc1ea54a6b55083547c6a932eef2
计划人id workid 是 int （即用户id）
计划人姓名 workname 是 string
开始时间 workstart 是 date
结束时间 workend 是 date
完成度状态 workstatus 是 int 1、已完成2、未完成3、未开始
标题 worktitle 是 string
内容 workcontent 否 string
进度 workspeed 否 string
目标金额 goal 是 int
正常返回结果
字段名 变量名 必填 类型 说明
返回信息 msg 是 string 成功
返回状态码 code 是 int 0
错误返回结果
字段名 变量名 必填 类型 说明
返回信息 msg 是 string 失败
返回状态码 code 是 int 1
